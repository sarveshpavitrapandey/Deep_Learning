# NLP Text Classification using WELFake Dataset

## Objective
To apply NLP preprocessing techniques and build a machine learning model for fake news classification.

## Techniques Used
- Tokenization
- Stopword Removal
- Stemming
- Lemmatization

## Vectorization
- TF-IDF
- CountVectorizer

## Model
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision, Recall, F1-score
- Confusion Matrix

## Dataset
WELFake Dataset (Kaggle)

## Result
TF-IDF performed better than CountVectorizer.
